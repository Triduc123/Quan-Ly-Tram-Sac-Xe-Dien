# Quan-Ly-Tram-Sac-Xe-Dien
Dự án E-Charge Station tập trung giải quyết bài toán di chuyển xanh thông qua giao diện trực quan, hỗ trợ người dùng nhanh chóng định vị và kiểm tra trạng thái trạm sạc nhanh 250kW trong bán kính gần nhất.
